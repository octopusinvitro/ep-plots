--- title: 3. Sparklines with figure and ul description: Applied to the ages of Northern Ireland politicians by term layout: page ---

The sparklists are in a ul. Each line is in a li.
Wrap every data value in each line inside a span and style it as a bar.
Then wrap all data values in that line inside another span to style the whole dataset.

A figure element is used to give semantic meaning:

"Usually [a figure]... is an image, an illustration, a diagram, a code snippet, or a schema that is referenced in the main text, but that can be moved to another page or to an appendix without affecting the main flow".

Hover to zoom.
