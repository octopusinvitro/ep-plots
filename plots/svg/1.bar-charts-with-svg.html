All the elements are drawn in an inline SVG. They can be styled with CSS from the site's CSS files and made interactive with JS. They are also accessible.

The parties and percentages, are made with the text element. The bars are made with the rect element. Each bar is grouped in its own group element g.

Classes applied directly to group elements through the class attribute won't apply, but you can still apply styles to group elements using the cascade, for example svg g { //styles }.
