EveryPolitician plots

Using web technologies (CSS, JS and SVG)

Visualizing data makes it easier to understand, analyze, and communicate. This site is the result of teaching myself how to use web technologies (HTML, CSS, JS, SVG, etc.) to make interactive plots. The first batch uses CSS, the second batch uses SVG, and a third batch is coming which uses a JS library called D3.js.

The dataset used for the plots is taken from the EveryPolitician project. Part of the idea behind this was also to explore the dataset's possibilities and limitations in terms of the information you can obtain from it, how difficult is to obtain, and how to use it in a meaningful way. For example, most of the data is discrete, like gender or ages.

Disclaimers: 1. This is a work in progress.
2. The data is fetched using AJAX, although it could be served by your favorite backend framework. EveryPolitician's Popolo files are close to 1Mb big and take around 1-2 seconds to download in a fast connection (55Mbps).
